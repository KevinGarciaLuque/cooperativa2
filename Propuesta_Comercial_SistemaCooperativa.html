1. Descripción General del Sistema

El Sistema de Gestión de Cooperativas es una aplicación web completa de arquitectura cliente-servidor (Full Stack), desarrollada con tecnologías modernas. Cuenta con dos grandes módulos: un Panel Administrativo completo y un Portal de Socio (perfil personal con vista de cuenta, aportaciones, préstamos y movimientos).

✅ Módulos funcionales implementados:
  • Autenticación segura con JWT y recuperación de contraseña por correo electrónico
  • Dashboard con estadísticas en tiempo real y gráficas interactivas
  • Gestión de Usuarios (altas, bajas, edición, foto de perfil, roles)
  • Gestión de Roles y Permisos granulares por módulo
  • Gestión de Cuentas de socios
  • Gestión de Préstamos (créditos, cuotas, seguimiento)
  • Gestión de Aportaciones (registro y seguimiento de aportaciones de socios)
  • Gestión de Pagos
  • Registro de Movimientos contables
  • Liquidaciones
  • Actividades de la cooperativa
  • Reportes exportables a Excel (ExcelJS)
  • Bitácora de auditoría (registro de acciones del sistema)
  • Gestión de Base de Datos: backups manuales y automáticos programados (cron)
  • Panel SuperAdministrador con configuración del sitio, logo y parámetros globales
  • Portal de Socio: vista personal de cuenta, movimientos y estado de préstamos
  • Subida de archivos (logos, fotos de usuario)
  • Sistema multirol: SuperAdmin, Admin, Operador, Socio
⚠️ Nota: El sistema actualmente no incluye integración con pasarelas de pago ni transferencias bancarias. Este módulo queda como funcionalidad futura.

4. Opciones de Comercialización

🔒 Política de Protección del Software: En ninguna modalidad se entrega el código fuente al cliente, salvo en la Opción C (Licencia Empresarial Exclusiva) con contrato legal notariado. El software se protege mediante compilación, ofuscación y acuerdo de confidencialidad (NDA). Esto garantiza que el cliente no puede replicar, revender ni redistribuir el sistema bajo ninguna circunstancia.

⭐ Opción A — SaaS en la Nube (Modalidad Recomendada)

El cliente accede al sistema a través de internet. Nunca recibe el código fuente. El desarrollador administra el servidor, las actualizaciones, los backups y la seguridad. Es la opción más segura para el proveedor y la más económica para el cliente.

Plan Alcance Mensual (USD) Anual (USD)
Plan Básico Hasta 100 socios, 2 administradores, hosting incluido, soporte por correo $150 – $200 $1,600 – $2,200
Plan Estándar Hasta 500 socios, 5 usuarios admin, soporte mensual incluido, backups automáticos diarios $280 – $380 $3,000 – $4,200
Plan Premium Socios ilimitados, usuarios ilimitados, soporte prioritario 48h, SLA garantizado, actualizaciones incluidas $500 – $700 $5,400 – $7,500

✅ Ventajas para el proveedor: El cliente nunca recibe el código. No puede replicarlo ni revenderlo. Vos mantenés el control total del software.

💡 Con solo 4 clientes en Plan Estándar se generan $1,120 – $1,520/mes de ingreso recurrente. El mercado salvadoreño tiene más de 500 cooperativas activas registradas ante el INSAFOCOOP.

💻 Opción B — Licencia On-Premise (Instalación en Servidor del Cliente, sin código)

Para clientes que necesitan el sistema instalado en su propio servidor (por política interna o regulatoria), se entrega únicamente el ejecutable compilado y ofuscado, nunca el código fuente legible. El cliente no puede leer, modificar ni redistribuir el software. Se firma un Contrato de Licencia de Uso Restringido que establece penalidades legales en caso de incumplimiento.

Modalidad Qué se entrega Precio (USD)
Licencia On-Premise Básica Build compilado + instalación en servidor del cliente + contrato de licencia de uso + 3 meses soporte $6,000 – $8,500
Licencia On-Premise Estándar Build compilado + personalización de marca + instalación + contrato + 6 meses soporte + actualizaciones $9,000 – $13,000
Licencia On-Premise Premium Build compilado + personalización completa + instalación + contrato + 12 meses soporte + 2 actualizaciones mayores $14,000 – $18,000

🔒 Protección: El frontend se entrega como archivos compilados (dist/ minificados). El backend se entrega ofuscado o empaquetado con herramientas que impiden su lectura. El contrato prohíbe expresamente la ingeniería inversa, copia o redistribución.

💡 Precio recomendado para empresa en El Salvador: $9,000 – $13,000 USD (Licencia On-Premise Estándar). Competitivo frente a software cooperativo regional.

🚨 Opción C — Licencia Empresarial con Código Fuente (Solo bajo contrato notariado)

Esta opción es exclusiva para grandes empresas o grupos corporativos que requieren acceso al código fuente por motivos de auditoría interna, personalización profunda o integración con otros sistemas empresariales. Requiere contrato notariado con cláusulas de confidencialidad, prohibición de redistribución y penalidades económicas por incumplimiento. No se recomienda para cooperativas individuales.

⚠️ Advertencia: Aunque el contrato protege legalmente, entregar código fuente siempre conlleva un riesgo residual. Solo se recomienda cuando el cliente tiene una razón técnica o legal válida y justificada, y cuando el precio refleja ese riesgo adecuadamente.

10. Notas y Consideraciones Finales

  • Los precios indicados son estimados de referencia en USD, orientados al mercado empresarial de El Salvador (economía dolarizada, sin riesgo cambiario).
  • El código fuente NUNCA se entrega en los planes SaaS ni On-Premise. Solo se entrega bajo contrato notariado en la Licencia Corporativa Exclusiva.
  • En la modalidad On-Premise, el frontend se entrega como archivos de producción compilados (dist/) y el backend como ejecutable o paquete ofuscado, lo que impide su lectura, modificación o redistribución.
  • Todo cliente firma un Contrato de Licencia de Uso con cláusulas de confidencialidad (NDA), prohibición de ingeniería inversa y penalidades por incumplimiento.
  • El sistema no incluye integración con pasarelas de pago ni banca en línea. Agregar ese módulo (ACH, banca salvadoreña) justificaría un incremento de $3,000 – $6,000 USD adicionales.
  • El código está construido con React 19, Vite 7 y Node.js en sus versiones más recientes, garantizando soporte, seguridad y escalabilidad a largo plazo.
  • Puede desplegarse en la nube (Railway, AWS, DigitalOcean) o en servidor propio del cliente.
  • Se recomienda incluir un contrato de mantenimiento anual de $800 – $1,500 USD para renovaciones, parches de seguridad y actualizaciones menores.
